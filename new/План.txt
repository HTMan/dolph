1. Схема расписаний:

Таблицу групп нужно будет выбирать из списка слева от нее.
Это будет сделано через горизонтальный flex box.
Список груп будет через вертикальный flex box.
Список занятий будет делиться на дни недели, если по ним что-то будет.
Это будет вертикальный flex box.
Каждый представленный день недели будет кнопкой, которая раскрывает график выбранного дня (время, тренер и т. п.)
Заголовки будут фиксированы сверху экрана.
Описание каждой группы также будет сверху.

2 Схема цен:

Цены будут также делиться 2 категории: группы и остальные услуги.
Группы также, как через план, будут выбираться.
Для каждой услуги указывается: для скольких взрослых и/или детей, продолжительность посещения, число посещений, цена.
Заголовки будут фиксированы сверху экрана.
Описание каждой группы также будет сверху.
Для услуги с НДС: услуга на кисл. коктейль указывает объем, цену
